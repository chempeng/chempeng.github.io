The Installation of Sci-hub Plugin

chempeng / 2017-05-21

A lawsuit isn’t going to stop [Sci-Hub], nor is there any obvious technical means. Everyone should be thinking about the fact that this is here to stay. – Peter Suber, Harvard University

本文简单介绍 sci-hub 浏览器插件的安装与使用，插件适用 chrome 浏览器或可以使用 chrome 插件的浏览器，如 360 浏览器。

  1. 下载插件并解压，插件下载地址 https://pan.baidu.com/s/1c1T9rDQ 。
  2. 在浏览器地址栏输入 chrome://extensions 。 ，进入浏览器扩展程序管理页面，其他浏览器类似。
  3. 勾选开发者模式，如果已安装旧版本 sci-hub 插件，需卸载老版本。
  4. 加载已解压扩展程序，选择插件解压后的目录 Sci-Hub。
  5. 重启浏览器时，如出现请停用以开发者模式运行的扩展程序的提示，选择取消。

插件安装成功后，可实现以下功能

  1. 直接点击插件图标，进入 sci-hub 主页。
  2. 在文献页面点击图标，如果 sci-hub 数据库拥有该文献，即可获得文献。
  3. 使用谷歌学术搜索出的文献，点击搜索链接，直接通过 sci-hub 尝试下载文献。
