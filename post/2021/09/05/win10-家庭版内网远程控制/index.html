win10 家庭版内网远程控制

chempeng / 2021-09-05

本文基于 Win10, 安卓手机或平板电脑（苹果手机或 Mac 电脑适用）

需求：在内网环境下，手机或平板通过微软远程桌面功能远程控制 Win10 家庭版

软件：

  • SuperRDP GitHub | 下载
  • RDPWrap GitHub | 下载
  • Remote Desktop v8 下载 | v10 下载

电脑端配置

Win10 家庭无法使用远程桌面功能，故通过软件绕过系统实现这一功能。

  1. 控制面板 -> 系统和安全 -> 允许远程访问（系统）-> 勾选允许远程协助连接这台计算机
  2. 安装运行 SuperRDP，使用说明见 GitHub
  3. 检测远程桌面是否启用成功，使用 RDPWrap 中 RDPCheck.exe 程序
    • 可登录，即启用成功
    • 拒绝访问，打开 RDPConf.exe，取消勾选 Single session per user，再尝试登录
  4. 设置 -> 系统 -> 电源和睡眠 -> 睡眠改为从不

远程端配置

  1. 下载安装 Remote Desktop v8 或 v10 版本
  2. 处于内网环境或通过 VPN 连接内网
  3. 添加 Desktop，PC name 填无线网络 ipv4 地址，User name 填微软登录账户和密码即可，点击即可远程访问
